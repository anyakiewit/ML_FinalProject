
# Human-Machine Mixed Text Detection
SemEval-2024 Task 8, Subtask C
A machine learning pipeline for detecting the boundary between human-written and machine-generated text in mixed documents. Given a text where the first segment is human-written and the second is machine-generated, the goal is to predict the word index at which the switch occurs.
This project was built for the MLP-2026 final project at the University of Groningen.

Requiered libraries can be found in the requirements.txt file.
